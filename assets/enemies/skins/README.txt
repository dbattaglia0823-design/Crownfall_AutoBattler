Enemy non-base skin sprite sheets live here.

Base enemy skins do not use this folder. They keep using the existing paths in SPRITE_SHEETS.
Every non-base enemy skin has its own explicit path in SKIN_SPRITE_SHEETS in js/data.js.

Use the same 6-frame horizontal sheet format as the base sprites:
idle, windup, attack, block, hit, downed.
